💪 FitTracker

Application mobile de suivi de musculation — légère, rapide, sans compte, sans pub.

Fonctionne comme une vraie app sur Android (PWA) tout en étant un simple site web hébergé gratuitement sur GitHub Pages.


Fonctionnalités


Séance quotidienne — saisie rapide des reps avec boutons +1 / +10 / −1 / −10
Exercices personnalisables — ajoute, modifie ou supprime tes exercices librement
Objectifs & progression — barre de progression par exercice, objectif quotidien configurable
Records personnels — détection automatique des PRs à chaque séance
Streak — compteur de jours d'entraînement consécutifs
Historique — toutes tes séances consultables et supprimables
Statistiques — graphique sur 7 jours, comparaison semaine passée, totaux
Suivi du poids — pesées mensuelles avec courbe d'évolution
Mode sombre — s'adapte automatiquement aux préférences du téléphone
Données locales — tout est stocké sur ton téléphone, aucun serveur
